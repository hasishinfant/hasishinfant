<summary><b>🐍 Want an animated snake eating your contribution graph? (optional, 2-minute setup)</b></summary>

<br/>

This turns your contribution graph into an animated snake that "eats" each green square. It runs automatically once a day via GitHub Actions, entirely inside your own <code>hasishinfant/hasishinfant</code> repo — no external service needed.

**Setup:**

1. In your `hasishinfant/hasishinfant` repo, go to **Settings → Actions → General** and allow all actions.
2. Create a new file at `.github/workflows/snake.yml` with the content below.
3. Push it — the Action runs automatically on the next day's schedule (or trigger manually from the Actions tab).
4. Add the image tag below anywhere in this README.

```yaml
name: generate animation
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: hasishinfant
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Then embed it with:

```md
<picture>
  <source media="(prefers-color-scheme: dark)" srcset="https://raw.githubusercontent.com/hasishinfant/hasishinfant/output/github-contribution-grid-snake-dark.svg" />
  <img alt="snake animation" src="https://raw.githubusercontent.com/hasishinfant/hasishinfant/output/github-contribution-grid-snake.svg" />
</picture>
```
